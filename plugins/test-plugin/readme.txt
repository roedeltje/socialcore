=== Test Plugin voor SocialCore ===
Bijdragers: SocialCore Team
Vereist: SocialCore 1.0+
Getest tot: SocialCore 1.0
Versie: 1.2.0

== Beschrijving ==

Dit is een eenvoudige test plugin om het plugin systeem van SocialCore te testen.

== Installatie ==

1. Upload de plugin naar /plugins/test-plugin/
2. Activeer de plugin via het admin dashboard
3. De plugin voegt automatisch HTML comments toe
